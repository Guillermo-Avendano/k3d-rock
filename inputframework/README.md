# k3d Rocket Solutions

## Introduction

- InputFramework 11.2

### Prerequisites
```bash
cd k3d-rock/inputframework
```

## Summary of commands

| Command | Description |
|:---|:---|
| ./inputframework.sh  install | Install InputFramework |
| ./inputframework.sh  remove  | Remove InputFramework |
